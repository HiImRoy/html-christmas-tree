# 🎄 3D 圣诞树 (手机电脑通用版)

这是一个可以直接在浏览器里打开的 3D 圣诞树。
不需要安装任何 APP，不需要复杂的配置，下载文件就能玩！

## 📂 怎么打开？

### 🖥️ 电脑用户 (Windows / Mac)
1.  下载本项目中的 `空圣诞树.html` 文件。
2.  **直接双击**这个文件。
3.  如果不自动跳转，右键点击文件 -> 选择 **Google Chrome** 或 **Edge 浏览器** 打开。

### 📱 手机用户 (Android / iOS)
1.  把 `空圣诞树.html` 文件发送到你的手机（通过微信、QQ、网盘等）。
2.  在手机上点击文件，选择 **“用浏览器打开”** (推荐 Chrome 或 手机自带浏览器，苹果打开html文件的方法请自行百度)。
3.  *提示：微信内置浏览器可能无法全屏，建议选择“在浏览器打开”体验更好。*

## 📸 怎么把照片挂上去？

你可以把家人、朋友或偶像的照片挂在树上。

### 方法一：我就想玩一下（最简单）
1.  打开圣诞树网页。
2.  点击底部的 **“上传照片”** 按钮。
3.  从你的相册里选几张照片，它们就会自动挂到树上了！
    *   *注意：这种方式刷新网页后照片会消失。*

### 方法二：我想做一个带照片的文件发给朋友（永久保存）
如果你想把照片“焊死”在树上，发给别人打开就能看到，请这样做：

1.  在电脑上，右键点击 `空圣诞树.html`，选择 **“打开方式” -> “记事本”** (或者任意文本编辑器)。
2.  按 `Ctrl + F` (查找)，输入：`const ossImageUrls`。
3.  你会看到下面这样的代码：
    ```javascript
    const ossImageUrls = [
        // 此处可自行放入图片链接
    ];
    ```
4.  把你准备好的**图片链接**粘贴进去(可以使用OSS对象存储得到自己的链接)，像这样：
    ```javascript
    const ossImageUrls = [
        "https://你的图片链接1.jpg",
        "https://你的图片链接2.png"
    ];
    ```
    *(注意：链接必须用双引号括起来，如果是多张图片，中间要用逗号隔开)*
5.  **保存文件**。
6.  现在你把这个 HTML 文件发给任何人，他们打开就能看到你设置好的照片了！

## 🎮 怎么玩？

*   **转动视角**：
    *   电脑：按住鼠标左键拖动。
    *   手机：单指在屏幕上滑动。
*   **看照片**：
    *   看到树上的照片了吗？**点击它**，镜头会自动拉近。
    *   再点一下旁边的空白处，就会退回来。
*   **隐藏菜单**：
    *   觉得按钮挡住视线了？双击屏幕背景，菜单就会消失/出现。

## ⚠️ 常见问题

**Q: 我双击打开是黑屏，什么都没有？**
*   请确保你的电脑连网了（因为特效需要加载网络资源）。
*   尝试换一个浏览器，推荐使用 **Chrome**、**Edge** 或 **Firefox**。IE 浏览器可能不支持。

**Q: 图片链接哪里来？**
*   你可以把图片上传到图床（如聚合图床、微博图床），或者复制你发在 QQ 空间/朋友圈图片的“原图链接”。
